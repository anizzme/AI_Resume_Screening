# AI Resume Screening System

## 📌 Description
An NLP-based system that ranks resumes based on job descriptions using similarity techniques.

## 🚀 Features
- Resume text preprocessing
- TF-IDF vectorization
- Cosine similarity-based ranking
- Automated candidate matching

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy

## ▶️ How It Works
1. Extract text from resumes  
2. Convert text into numerical vectors (TF-IDF)  
3. Compute similarity with job description  
4. Rank candidates based on similarity score  

## 📊 Key Highlights
- Automated resume screening process  
- Improved candidate matching efficiency
